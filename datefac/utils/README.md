# Utils Package

Purpose:
- Small reusable helpers such as artifact names, logging helpers, and run-state utilities.

Not responsible for:
- numbered-task orchestration

Place new files here when:
- the logic is small, reusable, and not domain-specific enough for another package

Category:
- source code

MinerU-first / table-first relation:
- utility helpers may support MinerU tasks indirectly, but this package is intentionally generic.
