### 4. Deterministic trial standardization heuristics
Use deterministic heuristics only.

Target metrics:
- 营业收入
- 归属母公司净利润 / 归母净利润
- 每股收益 / EPS
- P/E / PE
- P/B / PB
- EV/EBITDA
- ROE
- EBITDA

Year patterns:
- 2024A / 2025A / 2026E / 2027E / 2028E
- 2024 / 2025 / 2026 / 2027 / 2028

Rules:
- Normalize years with no suffix only conservatively:
  - do not force `2025` to `2025A` unless there is strong table context;
  - record a flag such as `year_suffix_missing`.
- Extract numeric values from rows where a metric keyword is found and the same table has year-like columns/cells.
- If mapping between values and years is ambiguous, do not invent values. Route to manual_review_candidate with flag `ambiguous_year_value_alignment`.
- Negative values are allowed for profit/EPS/PE in loss-making samples, but add flag `negative_value_present`.
- Do not mark outputs as trusted. This is a trial. Use route_recommendation values:
  - likely_core_metric_trial
  - manual_review_candidate
  - ignore
