# Tools Directory

Purpose:
- Runnable entry scripts, task launchers, and local helper scripts such as PowerShell environment checks.

Not responsible for:
- Deep reusable business logic that belongs inside `datefac/`.

Place new files here when:
- a task needs a direct executable entry point
- a local environment helper or one-shot launcher is needed

Category:
- runtime entry points and local utilities

MinerU-first / table-first relation:
- this directory contains the main MinerU benchmark runners and environment check/repair helpers used by the current benchmark chain.
