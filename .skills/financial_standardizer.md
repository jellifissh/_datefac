# Skill: 05 财务指标标准化

## 目标指标（8项）
1. 营业收入  
2. 归属母公司净利润  
3. 毛利率  
4. ROE  
5. 每股收益  
6. P/E  
7. P/B  
8. EV/EBITDA

## 当前关键能力
- 年份列识别：支持 `2024 / 2024A / 2025E / 2024年` 等形式。
- 表头修复：`header_repaired`，可在无年份列名时从前几行重建列头。
- 行标签扫描：前 8 列候选扫描（`metric_candidate_scan`）。
- 回退策略：`fallback_first_columns`。
- 归母净利润别名支持：
  - 归属母公司股东净利润
  - 归属于母公司股东的净利润
  - 归属于母公司所有者的净利润
  - 归母净利
  - 归母净利润
  - 母公司股东净利润
  - 归属于上市公司股东的净利润

## 防误抽规则
- 不把“营业收入同比”抽成“营业收入”。
- 不把“归母净利润同比/增速/增长率”抽成“归属母公司净利润”。
- 不把“扣非归母净利润”抽成“归属母公司净利润”。
- 不把“少数股东损益”抽成“归属母公司净利润”。

## 每次验证必看字段
- 抽取明细行数（`detail_rows`）
- 核心指标宽表非空指标数量
- 命中指标名称 / 未命中指标名称
- `header_repaired=True` 行数
- `label_detect_method` 分布
- 是否存在同比/增速/扣非/少数股东损益误抽

## 改动边界
- 优先做低风险增强（别名、标签定位、误抽过滤）。
- 不在本层做复杂混合表拆分，避免越界改上游逻辑。
